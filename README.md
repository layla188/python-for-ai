# 🌦️ Weather AI Analysis

A beginner AI engineering project built with Python that demonstrates how to collect weather data from an external API and analyze text using a Hugging Face pipeline.

---

## 📌 Project Overview

This project combines data collection and artificial intelligence in a simple workflow:

- Fetches live weather data using an external weather API.
- Processes the collected data with Python.
- Uses a Hugging Face Transformers pipeline for AI analysis.
- Saves generated outputs for later use.

This project was created as part of the **AI Engineer Starter Kit** mini project.

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/layla188/python-for-ai.git
```

Move into the project folder:

```bash
cd python-for-ai
```

Install the required packages:

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Project

Open the notebook:

```
weather_ai_analysis.ipynb
```

Run all cells from top to bottom.
